🏫 Liceos / Colegios

Catálogo institucional. Cada liceo se asigna a uno o más profesores y su logo aparece en el encabezado de los documentos generados.

💡
Cómo subir el logo del liceo
Los logos viven como archivos estáticos en assets/logos/colegios/<slug>.png. Después de crear el liceo aquí, sube el archivo PNG (mín. 256×256 px, fondo transparente) por git al repositorio. Si no existe el archivo se mostrará el logo por defecto.

🤖 Configuración IA — Gemini

Administra las API Keys y el modelo por defecto para todos los profesores.

🔑 API Keys del pool

El sistema elige una key al azar cada vez que un profesor genera material. Agrega 2–3 keys para distribuir la cuota gratuita. Las keys se guardan en Firestore y nunca se muestran completas en pantalla.
