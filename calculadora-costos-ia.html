Estrategias para maximizar margen
  • Usá Gemini Flash por defecto. Pro es 10-30× más caro. Reservá Pro solo para adecuaciones PIE.
  • Rate limits por plan: Basic 20 docs/mes/docente, Pro 60/mes, Enterprise 200 con alerta.
  • Cache de docs similares: ahorra ~90% en docs repetidos entre docentes.
  • Context caching Gemini: system prompt cacheado tiene 75% descuento.
  • BYOK Enterprise: el colegio grande paga su propia cuenta Google Cloud.
  • Alertas presupuesto Google Cloud: $50/$100/$200 USD por colegio, si dispara avisa por email.
